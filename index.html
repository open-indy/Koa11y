Koa11y

Koa11y is a desktop app that allows you to automatically detect accessibility (a11y) issues on webpages. View sample report generated by Koa11y.

System Requirements:

  • 130-170MB of avaliable disk space (depending on OS)
  • 120MB-200MB of avaliable memory (depending on OS)
  • Windows XP, Vista, 7, 8, and 10.
  • OSX 10.6 and above (64-Bit only).
  • Ubuntu 12+. Zorin 9+. Debian 8+. Other Linux versions, especially Debian based, may work but aren't supported.

Koa11y was created using Pa11y, UGUI, and NW.js.
